DATABASE stores the data.
ENTITY represents the data in Java.
REPOSITORY reads/writes the data.
SERVICE applies rules and logic.
CONTROLLER exposes endpoints to the outside world.
SWAGGER / TEST confirms everything is working.

(DATABASE → ENTITY → REPOSITORY → SERVICE → CONTROLLER → SWAGGER TEST)

SpEL = Spring Expression Language. 
It’s a small expression language 
built into Spring that allows you to 
dynamically evaluate expressions in annotations 
like @PreAuthorize or @PostAuthorize.

Tailwind CSS – A utility-based CSS framework that lets you style websites quickly using ready-made classes.

React – A JavaScript library for building fast, interactive user interfaces using components.

TypeScript – A typed version of JavaScript that helps catch errors and makes code easier to manage.

Node.js – A runtime that lets you run JavaScript on the server to build backend applications.
